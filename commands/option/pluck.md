# wp option pluck

Gets a nested value from an option.

### OPTIONS

&lt;key&gt;
: The option name.

&lt;key-path&gt;...
: The name(s) of the keys within the value to locate the value to pluck.

[\--format=&lt;format&gt;]
: The output format of the value.
\---
default: plaintext
options:
  - plaintext
  - json
  - yaml
\---
