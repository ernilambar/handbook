# wp super-admin list

Lists users with super admin capabilities.

### OPTIONS

[\--format=&lt;format&gt;]
: Render output in a particular format.
\---
default: list
options:
  - list
  - table
  - csv
  - json
  - count
  - ids
  - yaml
\---

### EXAMPLES

    # List user with super-admin capabilities.
    $ wp super-admin list
    supervisor
    administrator
