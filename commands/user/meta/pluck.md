# wp user meta pluck

Get a nested value from a meta field.

### OPTIONS

&lt;id&gt;
: The ID of the object.

&lt;key&gt;
: The name of the meta field to get.

&lt;key-path&gt;...
: The name(s) of the keys within the value to locate the value to pluck.

[\--format=&lt;format&gt;]
: The output format of the value.
\---
default: plaintext
options:
  - plaintext
  - json
  - yaml
