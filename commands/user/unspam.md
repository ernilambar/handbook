# wp user unspam

Removes one or more users from spam on multisite.

### OPTIONS

&lt;user&gt;...
: The user login, user email, or user ID of the user(s) to remove from spam.

### EXAMPLES

    # Remove user from spam.
    $ wp user unspam 123
    User 123 removed from spam.
    Success: Unspamed 1 of 1 users.
